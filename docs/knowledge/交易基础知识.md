# 交易基础知识

> OKX 永续合约（Perpetual Swap）核心概念，基于单向持仓 + 全仓模式。

---

## 1. 开仓（Open Position）

下单进场，建立仓位。

| 方向 | 含义 | 例子 |
|------|------|------|
| 做多（Long / Buy） | 赌涨 | 1U 买，1.1U 卖 → 赚 |
| 做空（Short / Sell） | 赌跌 | 1U 卖，0.9U 买回 → 赚 |

---

## 2. 持仓（Position / Hold）

开仓后持有的仓位，相关字段：

| 概念 | 字段 | 说明 |
|------|------|------|
| 仓位大小 | `size` | 持有多少张合约 |
| 开仓价 | `entry_price` | 进场成交均价 |
| 标记价格 | `mark price` | 当前市场公允价（用于计算盈亏和爆仓） |
| 未实现盈亏 | `unrealized pnl` | 浮动盈亏，仓位还在，没落袋 |
| 已实现盈亏 | `realized pnl` | 平仓后锁定的实际盈亏 |

```
未实现盈亏（做多）= (标记价 - 开仓价) × 数量
未实现盈亏（做空）= (开仓价 - 标记价) × 数量
```

---

## 3. 加仓（Add / Increase）

持仓中同方向追加，开仓均价会变：

```
新均价 = (旧数量 × 旧价格 + 新数量 × 新价格) ÷ (旧数量 + 新数量)
```

---

## 4. 减仓（Reduce）

平掉部分仓位。已平部分盈亏变成已实现盈亏，剩余仓位保持不动。

---

## 5. 平仓（Close / Exit）

全部清掉离场。开仓价与平仓价的差价 = 最终盈亏。

---

## 6. 止损 / 止盈（Stop Loss / Take Profit）

预设触发价，行情触碰自动平仓：

| 类型 | 意义 |
|------|------|
| 止损（Stop Loss） | 反向走到容忍极限，认赔离场，防止更大亏损 |
| 止盈（Take Profit） | 正向走到目标，锁定利润离场 |

例：100U 做多，止损 95U（-5%），止盈 110U（+10%）。

---

## 7. 爆仓（Liquidation）

杠杆放大亏损到本金不足以维持仓位时，交易所强制平仓。

```
爆仓价（做多）≈ 开仓价 × (1 - 1/杠杆)
```

| 杠杆 | 反向 ~1% | 即爆仓 |
|------|-----------|--------|
| 10x  | ~10%     |        |
| 50x  | ~2%      |        |
| 100x | ~1%      |        |

---

## 8. 手续费

OKX 永续合约费率（按名义价值计算，非保证金）：

| 类型 | 费率 |
|------|------|
| Maker（挂单） | 0.020% |
| Taker（吃单） | 0.050% |

一开一平 Taker 合计 0.10%。100x 杠杆下，手续费占本金的 10%，行情需跑 0.1% 以上才真正盈利。

---

## 一句话总结

```
开仓 → 持仓（浮动盈亏）→ 加仓/减仓 → 平仓（锁定盈亏）
                              ↓
                     止损认赔 / 止盈落袋
```
